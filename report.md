## Notes

- All data is stored in browser localStorage — no account required.
- Chrome extension connects to the live web app for timer sync and notifications.
- Extension blocks YouTube Shorts, Instagram Reels, TikTok, Facebook Reels, Twitter, Reddit, and Netflix in strict mode.
- Built with Next.js 16, React 19, TypeScript, Zustand 5, Tailwind CSS 4, and WXT Manifest V3.
