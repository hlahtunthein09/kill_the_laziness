# FocusFlow AI — အသုံးပြုသူ လက်စွဲ (မြန်မာဘာသာ)

> **FocusFlow AI** (ခရီးသည်၏ အမည် — Kill The Laziness) သည် developer နှင့် computer အသုံးပြုသူများအတွက် ရည်ရွယ်ထားသော **gamified productivity app** ဖြစ်ပါတယ်။ ဒီ app က သင့်ကို အဓိကကျတဲ့အလုပ်တွေအပေါ် focus လုပ်စေပြီး၊ social media နှင့် အခြား distraction တွေကို အနည်းငယ် ထိန်းချုပ်ကူညီပေးပါတယ်။

---

## ၁။ ဒီ app က ဘာလို့ လိုအပ်လဲ?

အင်တာနက်ခေတ်မှာ — YouTube Shorts, Instagram Reels, Facebook, Reddit, Twitter စတဲ့ platform တွေက အချိန်အများကြီးကို လိမ်ယူရယူလိုက်ပါတယ်။ FocusFlow AI က အောက်ပါ ပြဿနာတွေကို ဖြေရှင်းဖို့ ရည်ရွယ်ပါတယ် —

- 🎯 အလုပ်လုပ်နေစဉ် အခြား web page တွေဆီသို့ မှားမှား၍သွားခြင်း
- ⏰ အလုပ်အတွက် အချိန်မခွဲထုတ်နိုင်ခြင်း
- 📉 ရက်တိုင်း progress မမြင်ရလို့ စိတ်ပျက်ခြင်း
- 🏗️ ကိုယ့်ရဲ့ project များအတွက် တိကျသော goal မရှိခြင်း

---

## ၂။ အဓိက လုပ်ဆောင်ချက်များ (Core Features)

### 🗂️ ပရောဂျက် (Projects)

ပရောဂျက်က ကြီးမားတဲ့ ပန်းတိုင်တစ်ခုကို ကိုယ်စားပြုပါတယ်။ ဥပမာ —

- "Next.js App ရေးခြင်း"
- "Python Course ပြီးခြင်း"
- "Portfolio Website တည်ဆောက်ခြင်း"

သင်တိုင်း ပရောဂျက်အသစ်တစ်ခု ဖန်တီးနိုင်ပြီး၊ နောက်ပိုင်းမှာ အဲ့ပရောဂျက်အောက်မှာ အခန်းကဏ်းငယ်တွေ ထပ်ထည့်နိုင်ပါတယ်။

### 📋 အခန်းကဏ်းများ (Sub-pieces)

ပရောဂျက်ကြီးကို အသေးငယ်ဆုံး အစိတ်အပိုင်း (task) များအဖြစ် ခွဲထုတ်နိုင်ပါတယ်။

ဥပမာ — "Next.js App" ပရောဂျက်အောက် —

- "Hero Section ရေးခြင်း" — ၂၅ မိနစ်
- "Auth System တပ်ဆင်ခြင်း" — ၆၀ မိနစ်
- "API Route သုံးခြင်း" — ၄၅ မိနစ်

အခန်းကဏ်းတိုင်းမှာ **ကျသင့်သော မိနစ်** allocated minutes သတ်မှတ်ထားပါတယ်။

### ⏱️ Timer (အချိန်မှတ်တမ်း)

FocusFlow AI ၏ နှလုံးသား (heart) ဖြစ်ပါတယ်။

- **Project Timer** — ပရောဂျက်အပေါ် အချိန်အများဆုံး စုစုပေါင်း count up လုပ်ပါတယ်။
- **Sub-piece Timer** — သတ်မှတ်ထားသော မိနစ်များမှ တစ်ဆင့် count down လုပ်ပါတယ်။
- Start / Pause / Reset ခလုတ်များဖြင့် ထိန်းချုပ်နိုင်ပါတယ်။
- Sub-piece အချိန်ပြည့်သွားပါက အလိုအလျောက် ရပ်ပြီး complete အဖြစ် သတ်မှတ်ပါတယ်။

### 📊 Dashboard (ပင်မစာမျက်နှာ)

`/` စာမျက်နှာတွင် သင့်ရဲ့ focus အခြေအနေကို မြင်နိုင်ပါတယ် —

- စုစုပေါင်း ပရောဂျက်အရေအတွက်
- ယနေ့ focus ရည်မှန်းချက် (Daily Focus Goal)
- အစဉ်လိုက် focus ရက်များ (Streak)
- လက်ရှိ အဆင့် (Level)
- ခံတပ် (Dev-Fortress) visualization
- Quick Focus — အမြန်ဆုံး task တစ်ခုကို စတင်နိုင်ခြင်း

### ⚙️ Settings (ဆက်တင်များ)

`/settings` တွင် အောက်ပါအရာများကို ပြောင်းလဲနိုင်ပါတယ် —

- **Strict Mode** — distraction site များသို့ သွားရာတွင် blocker page သို့ပြန်ပို့ခြင်း
- **Warn Mode** — distraction site များတွင် သတိပေးချို့ချင်းပုံ သေးသော overlay ပြသခြင်း
- **Forbidden URLs** — ပိတ်ဆို့လိုသော website များ စာရင်း
- **Notifications** — desktop notification ဖွင့်ပိတ်
- **Sound** — timer complete / milestone အသံ ဖွင့်ပိတ်
- **Theme** — Light / Dark / System
- **Daily Focus Goal** — နေ့စဉ် focus မိနစ်ရည်မှန်းချက်
- **Distraction Log** — မကြာသေးမီက ပိတ်ဆို့ခံခဲ့ရသော site များကို ကြည့်ရှုခြင်း
- **Backup & Restore** — JSON file အဖြစ် data ထုတ်ယူခြင်း / ပြန်လည်သွင်းခြင်း

### 📅 Scheduled Focus Sessions (အချိန်စဉ်)

Settings မှတဆင့် အလုပ်လုပ်ရန် schedule တစ်ခု ထည့်နိုင်ပါတယ်။

ဥပမာ —

- "တနင်္လာနေ့ ည ၈:၀၀ နာရီကြာ ၃၀ မိနစ် Python study"
- "ဗုဒ္ဓဟူးနေ့ မနက် ၉:၀၀ နာရီကြာ ၂၅ မိနစ် API design"

အချိန်ရောက်လျှင် browser notification ဖြင့် သတိပေးပေးပါတယ်။

### 🏰 Dev-Fortress (Gamification)

Focus လုပ်လုပ်နေသလို XP ရပြီး၊ level တက်သလို ခံတပ်ကို တိုးတက်အောင်ဆောက်နိုင်ပါတယ်။

- မိနစ် ၁ မိနစ်အတွက် XP
- Sub-piece ပြီးမြောက်တိုင်း bonus XP
- Level တက်သလို ခံတပ် အဆင့်မြင့်လာမယ်
- Health bar ပါ ပြောင်သွားမယ်

## ၃။ Browser Extension (Chrome / Edge / Firefox)

FocusFlow AI သည် web app တစ်ခုဘဲ မကပါဘူး၊ browser extension တစ်ခုလည်း ရှိပါတယ်။

Extension က ဘာလုပ်ပေးလဲ —

- 🚫 YouTube Shorts, Instagram Reels, Facebook Reels, TikTok, Twitter, Reddit, Netflix စတဲ့ distraction site များကို ပိတ်ဆို့ပါတယ်
- 🔔 Tab ပိတ်ထားရင်လည်း background alarm ဖြင့် desktop notification ပေးပါတယ်
- 📟 Extension popup မှ timer status ကို မြင်နိုင်ပါတယ်
- ▶️ Popup မှတဆင့် timer Start / Pause / Reset လုပ်နိုင်ပါတယ်

Extension ကို load လုပ်နည်း —

1. `npm run build:ext` လုပ်ပါ
2. Chrome → `chrome://extensions` → Developer mode on → Load unpacked → `.output/chrome-mv3` ကို ရွေးပါ

## ၄။ အသုံးပြုနည်း အဆင့်ဆင့်

### ပထမအဆင့် — ပရောဂျက်တစ်ခု ဖန်တီးပါ

1. `/projects` သို့သွားပါ
2. "ပရောဂျက်အသစ်ထည့်မယ် (Add New Project)" ခလုတ်နှိပ်ပါ
3. ပရောဂျက်အမည် ထည့်ပါ
4. Save နှိပ်ပါ

### ဒုတိယအဆင့် — အခန်းကဏ်းထည့်ပါ

1. Project card အပေါ် "အခန်းကဏ်းအသစ် (Add Sub-piece)" ခလုတ်နှိပ်ပါ
2. အခန်းကဏ်းအမည် နှင့် မိနစ် (သာမန်အားဖြင့် ၂၅ မိနစ်) ထည့်ပါ
3. Save နှိပ်ပါ

### တတိယအဆင့် — Focus စတင်ပါ

1. Project card မှ "Focus လုပ်မယ်" ခလုတ်နှိပ်ပါ (သို့မဟုတ် dashboard မှ Quick Focus သုံး၍)
2. `/timer` သို့ ရောက်ပါလိမ့်မယ်
3. Start နှိပ်ပြီး focus လုပ်ပါ
4. Pause နှိပ်ပြီး ခဏရပ်နိုင်ပါတယ်
5. Reset နှိပ်ပြီး session ပြန်စနိုင်ပါတယ်

### စတုတ္ထအဆင့် — ပြီးမြောက်မှုကို ကြည့်ပါ

- Sub-piece အချိန်ပြည့်သွားပါက Session Summary card ပေါ်လာမယ်
- Dashboard မှာ ခံတပ် XP တက်သွားတာ မြင်နိုင်ပါတယ်

## ၅။ အသုံးဝင်သော သတိပေးချက်များ

- ⚡ **Burmese-first UI**: အသုံးပြုသူ interface အများစုကို မြန်မာဘာသာဖြင့် ရေးထားပါတယ်။ English subtitle ပါဝင်ပါတယ်။
- 💾 **Data Storage**: သင့်ဒေတာအားလုံးကို browser ၏ localStorage မှာ သိမ်းထားပါတယ်။ Backup / Restore feature ကို သုံးပြီး JSON file အဖြစ် backup လုပ်ထားနိုင်ပါတယ်။
- 🌙 **Dark Mode**: Settings မှ theme ကို Light / Dark / System ဟု ပြောင်းနိုင်ပါတယ်။
- 🔔 **Notification**: Browser notification permission ရယူထားမှ desktop notification ရပါမယ်။

## ၆။ ဘာအတွက် အထူးကောင်းမလဲ?

| ပြဿနာ | FocusFlow AI ဘယ်လိုကူညီမလဲ |
|---|---|
| Social media က လူကိုဆွဲဆောင်နေတယ် | Forbidden URLs + extension blocker ဖြင့် ပိတ်ဆို့ပေးတယ် |
| အလုပ်အတွက် အချိန်မခွဲနိုင်ဘူး | Sub-piece timer ဖြင့် Pomodoro-style focus sessions |
| Motivation မရှိဘူး | Dev-Fortress + XP + Streak ဖြင့် gamification |
| ဘယ် progress ရောက်နေလဲ မသိဘူး | Dashboard + daily goal + session summary |
| Data ပျက်သွားမှာကို စိုးရိမ်တယ် | Backup / Restore JSON |

## ၇။ အမြင်ချင်းဖြစ်နိုင်သော မေးခွန်းများ

**Q: FocusFlow AI က အခမဲ့လား?**
A: လက်ရှိ local-only version မှာ အခမဲ့ပါ။ Data အားလုံး သင့်ကွန်ပျူတာမှာဘဲ သိမ်းပါတယ်။

**Q: Mobile phone မှာ အသုံးပြုလို့ရလား?**
A: လက်ရှိ version မှာ browser extension မရှိသေးသလို၊ web app က responsive ဖြစ်ပါတယ်။

**Q: အချိန်ပြည့်သွားရင် အလိုအလျောက် ပြီးသွားလား?**
A: ဟုတ်ပါတယ်။ Sub-piece timer ဟာ ၀ သို့ရောက်လျှင် အလိုအလျောက် ရပ်ပြီး complete အဖြစ် သတ်မှတ်ပါတယ်။

**Q: Project တစ်ခုပြီးသွားရင် ဘာဖြစ်လဲ?**
A: Project အောက်မှ sub-pieces အားလုံး complete ဖြစ်သွားရင် project status လည်း completed ဖြစ်သွားပါတယ်။
